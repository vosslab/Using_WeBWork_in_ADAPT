WeBWorK's power is not a single feature, but a stable combination: algorithmic question setup, structured answer checking, and readable authoring of the student prompt.

Core features (author view)

  • PGML for prompt authoring (recommended): readable markup, math delimiters, and answer blanks.
  • MathObjects for robust parsing and checking of student answers (numeric, formulas, lists).
  • Randomization primitives (e.g., random()) to generate variants while keeping the scoring rules identical.
  • Multiple answer interaction types via macros (numeric answers, formulas, radio buttons, etc.).

Core features (student view)

  • Immediate feedback on correctness.
  • Consistent formatting for mathematical expressions.
  • Hints and solutions when enabled by the instructor.

Features that matter in sciences

  • Prompts that mix words, values, and math without fragile escaping.
  • Tables for measurement data (use table helpers rather than layout hacks).
  • Clean embedding of images/figures when needed (via command substitution).
  • Links to external references or built-in help links when appropriate.
